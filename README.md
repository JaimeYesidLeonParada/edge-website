# EDGE - Official Website

## Description
Frontend repository for the official website of EDGE, a startup dedicated to hardware development, Edge AI, and robust IoT monitoring solutions. This site serves as the primary landing page, product showcase, and contact portal for potential clients and partners.

## Tech Stack
* **HTML5**: Semantic structure.
* **CSS3**: Modern layout using Flexbox and CSS Grid, fully responsive.

## Project Structure
* `index.html`: Main landing page featuring the Hero section.
* `product.html`: Detailed overview of core technologies (Advanced Sensors, Edge Computing, Connectivity).
* `contact.html`: Contact information and inquiry form.
* `css/styles.css`: Global styling and responsive design rules.
* `assets/`: Directory for images, logos, and media files.

## How to Run
This is a static website. To view it locally:
1. Clone this repository.
2. Open `index.html` in any modern web browser.
No local server or build process is required.
